# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).
This project does not use semantic versioning — releases are date-based tags on `main`.

## [Unreleased]

### Added

- GitHub issue templates (bug report, feature request, `config.yml`)
- Pull request template (`.github/PULL_REQUEST_TEMPLATE.md`)
- `SUPPORT.md` with links to community help resources
- `.editorconfig` for consistent editor settings across contributors

### Changed

- README: full persona table (38 personas), badges, TOC, compatibility matrix
- CONTRIBUTING: added branch naming conventions and Conventional Commits guidance
- CODE_OF_CONDUCT: linked to full Contributor Covenant v2.1 text
- SECURITY: added private vulnerability reporting link and response timeline

## [0.2.0] - 2026-03-19

### Added

- Auto-skill router rule for Claude Code and Cursor (no slash commands required)
- CI, deploy-pages, labels-and-comments, and maintenance GitHub Actions workflows
- GitHub Pages site under `site/`

### Changed

- Expanded persona coverage to 38 roles
- README shortened and restructured with explicit tool-by-tool usage guidance

## [0.1.0] - 2026-03-18

### Added

- Initial repository setup with multi-persona directory structure
- Role-based `.claude` and `.cursor` persona packs
- Core open-source files: README, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY, LICENSE, CHANGELOG
